# Agnira MUN 2025 Website

## Overview
A sophisticated, modern website for Agnira MUN x EPS MUN 2025 conference featuring:

### Design Features
- **Dark Theme**: Maroon (#8B0000, #A52A2A) and Golden (#FFD700, #FFA500) color palette
- **Matrix Animation**: Animated background with falling characters spelling "AGNIRAMUN01"
- **Modern Aesthetics**: Sleek gradients, smooth transitions, and professional typography
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices

### Sections
1. **Hero** - Landing section with event details and call-to-action
2. **About** - Mission, vision, and conference features
3. **Committees** - UNGA DISEC, UNCSW, Lok Sabha, and International Press Corps
4. **Messages** - Leadership messages (Chairperson, Principal, Founders)
5. **Venue** - East Point School details with embedded map
6. **Registration** - Pricing and registration process
7. **Contact** - Contact information and footer

### Technical Stack
- React 18 + TypeScript
- Vite build tool
- Tailwind CSS
- Lucide React icons
- Custom fonts: Playfair Display (headings) + Inter (body)

### Key Interactions
- Smooth scroll navigation
- Hover effects with scale and shadow animations
- Active section tracking in navigation
- Mobile-responsive hamburger menu
- Matrix rain background animation

### Color Palette
- Primary Dark: #1a0a0f (deep maroon-black)
- Maroon: #8B0000, #A52A2A
- Golden: #FFD700, #FFA500
- Text: Gray scale for hierarchy

### Optimization
- SEO meta tags included
- Fast loading with optimized assets
- Accessibility considerations
- Custom scrollbar styling
